Aula: Negrito e Itálico no HTML

1. Usando <b> para negrito

O elemento <b> é usado para destacar o texto visualmente, deixando-o em negrito, mas sem indicar importância.

Exemplo: <p>Este texto está em <b>negrito</b>.</p>

Este texto está em negrito.

2. Usando <strong> para negrito com ênfase

O elemento <strong> também deixa o texto em negrito, mas indica que ele é importante no conteúdo.

Exemplo: <p>Este texto é <strong>importante</strong>.</p>

Este texto é importante.

3. Usando <i> para itálico

O elemento <i> deixa o texto em itálico, apenas como estilo visual.

Exemplo: <p>Palavras em <i>itálico</i> podem destacar citações.</p>

Palavras em itálico podem destacar citações.

4. Usando <em> para ênfase em itálico

O elemento <em> também deixa o texto em itálico, mas indica que o conteúdo tem uma ênfase especial.

Exemplo: <p>Você <em>realmente</em> precisa estudar.</p>

Você realmente precisa estudar.

Resumo

  • <b>: apenas negrito visual.
  • <strong>: negrito com ênfase/semântica.
  • <i>: apenas itálico visual.
  • <em>: itálico com ênfase/semântica.
